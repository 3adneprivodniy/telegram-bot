# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Telegram Bot

Node.js Telegram-бот з ІІ-обробкою: текст → Claude (Anthropic), картинки → DALL-E 3 (OpenAI), фото/скриншоти → Claude Vision, реал-тайм дані → зовнішні API.

## Команди

```bash
npm install

# Локальний запуск (тільки якщо Railway не працює — інакше дублювання відповідей)
node bot.js

pkill -f "node bot.js"

# Деплой — пуш в main, Railway деплоїть автоматично
git push origin main
```

## Архітектура

Весь бот — один файл `bot.js`. Порядок обробки вхідного текстового повідомлення:

1. `isImageRequest()` → DALL-E 3 генерація, зберігає промпт у `lastImagePrompt`
2. `isImageEditRequest()` + `lastImagePrompt.has(chatId)` → об'єднує старий промпт з новим, перегенерує через DALL-E 3
3. `isRealTimeRequest()` → `fetchRealTimeData()` підтягує курси/крипту/новини, додає в контекст Claude
4. Інакше → `anthropic.messages.create()` з системним промптом + історією чату

Фото (`bot.on('photo')`) — окремий обробчик, завжди через Claude Vision, в історію не додається.

Прогрес-повідомлення: одне повідомлення надсилається одразу, потім редагується через `setInterval`. Для картинок — видаляється перед надсиланням фото.

## Персистентне сховище

Два JSON-файли (в `.gitignore`, не комітяться):
- `lastImagePrompts.json` — останній промпт картинки per chatId
- `seenUsers.json` — множина chatId, які вже бачили `/start`

## Пам'ять чату

- `chatHistory: Map<chatId, messages[]>` — в пам'яті, скидається при перезапуску
- `MAX_HISTORY = 25`
- `INACTIVITY_TIMEOUT = 120_000` — автоочистка після 120 сек неактивності

## Реал-тайм дані

| Тип | API | Ключ |
|---|---|---|
| Курси валют (офіційний курс НБУ) | bank.gov.ua/NBU_Exchange/exchange_site | не потрібен |
| Крипта (будь-яка монета) | CoinGecko simple/price + search | не потрібен |
| Новини | Google News RSS (UA: `hl=uk&gl=UA`) | не потрібен |
| Погода (поточна) | Open-Meteo API + геокодинг | не потрібен |

Дані інжектуються в повідомлення як `[Актуальні дані на сьогодні: ...]` перед надсиланням в Claude.

### Логіка визначення крипти

`CRYPTO_ID_MAP` — словник з 30+ монет (аліас → CoinGecko ID). Якщо монета не знайдена в словнику, бот робить запит до `/api/v3/search` CoinGecko. Показує USD, UAH, EUR та зміну за 24г.

### Логіка визначення валют

`CURRENCY_KEYWORDS` — словник ключових слів → код валюти (USD, EUR, GBP, PLN, CHF, JPY, KZT, CZK). Якщо конкретна валюта визначена — показує тільки її курс до ₴. Інакше — USD, EUR, GBP, PLN, CHF, JPY.

## Команди бота

| Команда | Дія |
|---|---|
| `/start` | Перший раз: повне привітання. Повторно: короткий ответ |
| `/commands` | Список можливостей та команд |
| `/end` | Завершити розмову, очистити історію |
| `/clear` | Очистити історію чату |
| `/myid` | Показати свій chat_id |

## Деплой

Бот задеплоєний на Railway (автозапуск). **Ніколи не запускай локально одночасно з Railway** — два екземпляри з одним токеном викликають дублювання відповідей. SSH для GitHub налаштований через `~/.ssh/config` з ключем `~/.ssh/github`.
